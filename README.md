# markdown parser

this is sample of wasm markdown parser

## working versions

* cargo cargo 1.54.0-nightly
* wasm-bindgen 0.2.58
* ndoe ^v10.xx

## build

```bash
cargo install wasm-bindgen-cli
yarn
yarn build
```

## run

```bash
yarn serve
```
