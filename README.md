# markdown parser

this is sample of wasm markdown parser

## working versions

* cargo 1.41.0-nightly
* wasm-bindgen 0.2.58
* ndoe v10.17.0

## build

```bash
cargo install wasm-bindgen-cli
npm i
npm run build-debug
```

## run

```bash
npm run serve
```
